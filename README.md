# 📌 Activity Log

Este repositório **não contém código de produção**, ele existe apenas para registrar minha atividade de desenvolvimento.

## Por que ele existe?

O perfil do GitHub [luanmvcosta-engeman](https://github.com/luanmvcosta-engeman) é dedicado exclusivamente ao meu trabalho na **[Engeman](https://www.linkedin.com/company/engeman/)**. Como boa parte do meu dia a dia acontece em repositórios privados da empresa — commits, features, PRs — criei uma automação simples que, a cada commit feito, gera um registro espelho neste meu repositório principal, justamente para manter o histórico bonitinho no meu perfil do GitHub, contendo apenas **data/hora e nome do repositório envolvido** (sem expor código, mensagens de commit ou qualquer informação sensível da empresa).

O objetivo é manter um histórico consolidado da minha atividade como desenvolvedor na Engeman, já que grande parte do trabalho ocorre em repositórios privados e não fica visível publicamente.

## O que você vai encontrar aqui

- **`activity.log`** - um registro cronológico simples (data/hora + nome do repositório), sem nenhum conteúdo do código em si.
